\# Week 1 – Python, Motion, and PID Fundamentals



\## What I learned

\- How to simulate motion using time steps (dt) and simple physics

\- How closed-loop control works using error feedback

\- How P, I, and D terms affect system behaviour differently



\## Key files

\- day1\_basics.py – Python fundamentals

\- day2\_motion\_sim.py – Time and motion simulation

\- day3\_robot\_class.py – Robot1D class with acceleration

\- pid.py – PID controller implementation

\- day4\_pid\_test.py – Testing P, PI, and PID control



\## Notes

\- P responds to current error

\- I removes steady-state error by accumulating past error

\- D damps fast changes and reduces overshoot
